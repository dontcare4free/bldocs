String manipulation and processing
==================================

.. function:: strstr(%haystack, %needle)

   Searches for %needle in the %haystack.

   Returns: The index in %haystack where %needle begins, or -1 if it wasn't found.

   Thanks to: Sponge (6408)

.. function:: strlen(%str)

   Returns: The length of %str.

.. function:: strlwr(%str)

   Returns: %str in all lower-case.

   Thanks to: Xalos (11239)

.. function:: strupr(%str)

   Returns: %str in all upper-case.

   Thanks to: Xalos (11239)
